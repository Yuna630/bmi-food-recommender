# BMI & Meal Recommender

A Python application that calculates BMI, classifies body type, and recommends meals based on weight goals and food allergies.

## Features

- BMI calculation
- Body type classification
- Gender BMI comparison
- Food allergy handling
- Personalized meal recommendations
- Input validation

## Technologies

- Python
- Functions
- Dictionaries
- Lists
- Conditional Logic
- User Input Validation

## How to Run

python bmi_food_recommender.py

## Concepts Demonstrated

- Modular Programming
- Data Structures
- Input Validation
- Decision Making
- Personalized Recommendation Systems
